> Una página web interactiva que digitaliza un compendio histórico sobre el uso medicinal del azufre en seis civilizaciones ancestrales, basado en una profunda investigación de fuentes primarias como el Papiro de Ebers, los tratados de Dioscórides, el Charaka Samhita, el Bencao Gangmu y los códices mesoamericanos.

---

## 📜 Descripción del Proyecto

Este proyecto presenta, en formato de sitio web, un **manual técnico** que recopila y organiza las fórmulas, dosis, métodos de preparación y aplicaciones del azufre medicinal tal como fue utilizado por distintas culturas a lo largo de la historia.

La información proviene de una investigación exhaustiva de textos clásicos y tratados antiguos, traducidos y analizados para extraer las instrucciones precisas que los sabios de cada civilización legaron sobre el uso del azufre. La web no solo expone los datos, sino que los presenta con un diseño inmersivo que evoca manuscritos y grimorios antiguos, facilitando la consulta técnica y la apreciación histórica.

> ⚠️ **Advertencia importante:** Las fórmulas aquí descritas son reproducciones históricas con fines de estudio antropológico y farmacognóstico. Muchas implican la ingestión de compuestos sulfurados que, sin el procesamiento tradicional exacto, son altamente tóxicos. **No deben replicarse sin supervisión experta.**

## 🏛️ Fuentes de Información

La investigación se basa en las siguientes fuentes primarias y secundarias:

| Cultura | Fuente Primaria | Período Aprox. |
| :--- | :--- | :--- |
| **Mesopotamia** | Tablillas cuneiformes de la biblioteca de Asurbanipal (series terapéuticas) | 2000–500 a.C. |
| **Antiguo Egipto** | Papiro de Ebers (columnas 56, 60, 94) | 1550 a.C. |
| **Grecia y Roma** | *De Materia Medica* de Dioscórides; *Historia Natural* de Plinio el Viejo; *De Medicina* de Celso | Siglo I d.C. |
| **India Ayurvédica** | *Charaka Samhita*; *Rasa Ratna Samuchaya* de Vagbhata | 600 a.C.–Siglo XIII |
| **China Tradicional** | *Shennong Bencao Jing*; *Bencao Gangmu* de Li Shizhen | Siglo II a.C.–1578 |
| **Mesoamérica y Andes** | *Historia general de las cosas de Nueva España* de Sahagún (Códice Florentino); *Comentarios Reales* de Inca Garcilaso de la Vega | Siglo XVI |

## 📚 Estructura del Contenido

La página web se organiza en **seis capítulos culturales** más **apéndices técnicos**:

1. **Mesopotamia**
   - Fumigante purificador de habitación (*Bīt rimki*)
   - Ungüento contra la sarna (*epqu*)

2. **Antiguo Egipto**
   - Ungüento oftálmico para el tracoma (Receta Eb. 348)
   - Emplasto para forúnculos (Receta Eb. 108)

3. **Grecia y Roma**
   - Electuario oral para el asma (según Dioscórides)
   - Bálsamo «Acopa» para dolores articulares (según Plinio)
   - Tratamiento para sarna y tiña (según Celso)

4. **India Ayurvédica**
   - Proceso de purificación *Shodhana* con leche de vaca
   - Ciclo de rejuvenecimiento *Gandhaka Rasayana*
   - Pasta para enfermedades cutáneas severas (*Kushtha*)

5. **China Tradicional**
   - Purificación con tofu (*Doufu Zhi Liuhuang*)
   - Píldoras *Liuhuang Wan* para disfunción Yang y asma fría
   - Inhalación de emergencia por síncope

6. **Mesoamérica y Andes**
   - Emplasto mexica para huesos quebrados
   - Vapor de *Temazcalli* para dolores de nervios
   - Cataplasma inca para la uta (leishmaniasis)

7. **Apéndices Técnicos**
   - Tabla comparativa de medidas universales (dosis, mezclas, vehículos)
   - Métodos de purificación del azufre en las distintas culturas

## 🎨 Diseño y Experiencia de Usuario

El sitio está diseñado para sumergir al visitante en una atmósfera de sabiduría antigua, combinando legibilidad moderna con una estética clásica.

- **Paleta de colores:** Tonos pergamino, dorados sulfúreos y marrones cálidos.
- **Tipografía:** `Georgia` y `Cinzel` (esta última para títulos, evocando inscripciones clásicas).
- **Iconografía:** Uso del símbolo alquímico del azufre `🜂` como elemento decorativo y de navegación.
- **Navegación fija:** Barra superior con enlaces directos a cada cultura, con resaltado automático de la sección activa al hacer scroll.
- **Tarjetas de recetas:** Cada fórmula se presenta en una tarjeta visual independiente, con viñetas decorativas y resaltado de dosis exactas.
- **Citas destacadas:** Bloques estilizados para las frases textuales de los antiguos sabios.
- **Totalmente responsivo:** Adaptable a móviles, tablets y escritorio, con estilos de impresión optimizados.
- **Sin dependencias externas:** Una única página HTML autocontenida con CSS y JavaScript vanilla.

## 🚀 Cómo Usar

1. Clona el repositorio o descarga el archivo `index.html`.
2. Ábrelo en cualquier navegador moderno (Chrome, Firefox, Safari, Edge).
3. Navega por las culturas usando la barra superior o haciendo scroll.
4. Consulta las recetas, dosis y métodos de purificación.

No se requiere instalación, servidor ni conexión a internet (salvo para fuentes de Google Fonts, que se cargan opcionalmente; la tipografía base es del sistema).
